# 风控平台V1.3.2-需求文档

# 需求背景

:::
**问题**：黑产攻击手段多样（撞库、垃圾注册）、风险场景复杂（异地/新设备登录）、处置方式粗放（一刀切拦截），导致安全与体验难以平衡。
:::
:::
**目标**：构建端云协同的智能风控体系，形成可解释、可配置、可追踪的闭环风控平台。

1. 梯度化处置  

   实现风险梯度打分，降低对正常用户的影响。

2. 策略热生效  

   支持后台配置、快速启停，策略开启后 5 分钟内完成线上防护生效。

3. 场景全覆盖  

   覆盖登录、注册、支付三大核心场景，打通风险识别、行为分析、策略处置和结果回溯的闭环防护。

4. 数据驱动运营  

   通过安全概览、风控日志、风险行为分析帮助业务方快速识别风险漏洞，并推动风控策略逐步启用。
:::

**页面说明**

![image](https://docs-body.xiaoshanim.com/res/VXgNlD85J8ozj8R7/img/55bbcb96-6506-4212-b323-7032a06e1ec7.png?Expires=1778318540&OSSAccessKeyId=file-s3-gateway-ak&Signature=ZPhpLvurvNuzQYdHu0TB33JVLhs%3D)

1.  **页面权限：**当且仅当登录账号同时具有 “当前页面权限” 和 “项目权限”时，可见该页面的导航菜单
    
2.  **页面入口：**游戏工具-风控管理--白名单管理
    
3.  **页面结构：**操作区域包括页眉筛选、筛选查询、查询结果
    
4.  **页眉标题：**
    

*   主标题：白名单管理
    
*   副标题：可针对账号、设备或IP设置风控白名单，加入白名单的对象不再触发风控
    

1.  **页签切换：**支持切换账号/设备/IP，默认选中账号
    
2.  **筛选-生效对象：**支持按生效对象（账号/设备/IP）进行筛选
    
    *   标题：根据页签选中项，切换后同步更新，标题展示为账号/设备
        
    *   占位符：**根据页签选中项，切换后同步更新**，展示为请输入账号名或SDKID/请输入设备ID/请输入IP地址
        
    *   边界值：最大允许输入200字符，超出时截断输入
        
    *   默认值：默认无输入项，展示占位符
        
    *   匹配规则：子串匹配/包含匹配
        
3.  **筛选-生效状态：**支持按生效状态进行筛选
    
    *   标题：固定为生效状态
        
    *   交互效果：点击后展开/收齐 单选下拉菜单
        
    *   枚举值：下拉菜单选项包括生效中/已失效/未开始/全部
        
    *   默认值：默认为全部
        
    *   选中态：选中后展示已选中的选项
        
4.  **筛选-查询：**点击后提交查询，列表区域loading加载
    
5.  **筛选-重置：**点击后清空所有筛选项，但不立即触发查询
    
6.  **列表**
    
    *   标题：根据页签选中项，切换后同步更新，展示为账号白名单/设备白名单/IP白名单
        
    *   默认状态：默认展示全部名单数据
        
    *   排序规则：默认**按最近操作时间降序排序**
        
    *   异常状态：包括查询为空状态、查询失败状态、默认状态
        
    *   字段-生效对象
        
        *   页签选择账号时：展示SDKID和账号名两列
            
        *   页签选择设备时：展示设备ID
            
        *   页签选择IP时：展示IP地址
            
        *   交互效果：支持点击icon快速复制到剪切板
            
        *   展示规则：数据超出列宽时，使用"..."缺省表示
            
    *   字段-账号名
        
        *   只展示设置过的账号名，**不展示.dsp等自动生成的不对外的账号名**
            
        *   **账号名数据需要更新**，例如导入时只有SDKID，后续玩家绑定了账号名后，后台数据需要补充账号名
            
    *   字段-生效状态
        
        *   枚举值包括：生效中、已失效、未开始
            
    *   字段-生效时间
        
        *   数据格式：yyyy-mm-dd hh:mm:ss
            
    *   字段-失效时间
        
        *   数据格式：yyyy-mm-dd hh:mm:ss
            
    *   字段-添加原因
        
        *   展示规则：数据超出列宽时，使用"..."缺省表示
            
        *   数据来源：服务端传入，首次添加白名单时填入
            
    *   字段-最近操作人：
        
        *   数据来源：服务端传入
            
        *   数据更新：**新增和编辑更新后，同步更新数据**
            
        *   数据格式：形如wangjian02
            
    *   字段-最近操作时间：
        
        *   数据来源：服务端传入
            
        *   数据更新：**新增和编辑更新后，同步更新数据**
            
        *   数据格式：yyyy-mm-dd hh:mm:ss
            
    *   操作-编辑：点击后展开编辑弹窗，交互细节见下文章节“交互流程”
        
    *   操作-删除：点击后触发二次确认弹窗，交互细节见下文章节“交互流程”
        
7.  新增白名单：点击后展示配置弹窗，交互细节见下文章节“交互流程”
    
8.  批量导入：点击后展开右侧配置抽屉，交互细节见下文章节“交互流程”
    

**手动添加**

1.  **流程-新增白名单：**
    

![image](https://docs-body.xiaoshanim.com/res/VXgNlD85J8ozj8R7/img/9940ce4d-fb90-4350-8c0b-a9b41f04bc74.png?Expires=1778318540&OSSAccessKeyId=file-s3-gateway-ak&Signature=Rbwf6XylmHBG0%2FXsxjxzLWZfbmM%3D)

*   弹窗类型：包括三种，同步页签选中状态，包括账号设备IP三种
    
*   弹窗标题：新增账号白名单、新增设备白名单、新增IP白名单
    
*   新增账号白名单：
    
    *   账号类型选择：下拉单选；默认选中账号名；**枚举值包括账号名、SDKID**；必填
        
    *   账号信息输入：**占位符展示联动账号类型选择**，分别展示为“请输入账号名”/“请输入SDKID”；最大允许输入100字符；必填
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
    *   添加原因：必填，最大允许输入200字符；占位符“请输入添加白名单的原因”
        
*   新增设备白名单：
    
    *   设备信息输入：占位符展示为“请输入设备ID”；最大允许输入100字符；必填
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
    *   添加原因：必填，最大允许输入200字符；占位符“请输入添加白名单的原因”
        
*   新增IP白名单：
    
    *   IP信息输入：占位符展示为“请输入IP地址”；最大允许输入100字符；必填
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
    *   添加原因：必填，最大允许输入200字符；占位符“请输入添加白名单的原因”
        
*   确认按钮：点击后触发校验，期间loading，校验通过后展示二次确认弹窗：
    
    *   填写完整性校验：若未填写完成，则对应为填写项输入框标红
        
    *   生效对象不存在：信息输入框标红，并且toast提示 账号名不存在/SDKID不存在/设备ID不存在/IP不存在
        
    *   信息重复性校验：若生效对象已存在，则toast提示 该账号已填加白名单/该设备已填加白名单/该IP已填加白名单
        
*   二次确认弹窗：
    
    *   标题：提示
        
    *   描述：加入白名单的对象，在生效期间内不再触发风控。请确认是否添加
        
*   取消按钮&关闭按钮：点击后关闭，无二次确认；关闭后不保存上一次的填写内容

2.  **流程-编辑白名单**
    

*   弹窗类型：包括三种，同步页签选中状态，包括账号设备IP三种
    
*   弹窗标题：编辑账号白名单、编辑设备白名单、编辑IP白名单
    
*   编辑账号白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   编辑设备白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   编辑IP白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   二次确认弹窗：
    
    *   标题：提示
        
    *   描述：加入白名单的对象，在生效期间内不再触发风控。请确认是否更新生效时间
        
*   取消按钮&关闭按钮：点击后关闭，无二次确认

4.  **流程-编辑白名单**
    

*   弹窗类型：包括三种，同步页签选中状态，包括账号设备IP三种
    
*   弹窗标题：编辑账号白名单、编辑设备白名单、编辑IP白名单
    
*   编辑账号白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   编辑设备白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   编辑IP白名单：
    
    *   配置项：仅允许配置时间，其他信息均为只读
        
    *   生效时间：必填，开始时间和结束时间均为必填；占位符“请选择生效时间”和“请选择失效时间”；最大允许选择时长不超过1年；不允许选择过去时间
        
*   二次确认弹窗：
    
    *   标题：提示
        
    *   描述：加入白名单的对象，在生效期间内不再触发风控。请确认是否更新生效时间
        
*   取消按钮&关闭按钮：点击后关闭，无二次确认

5.  **流程-删除白名单**
    

*   交互效果：点击后触发二次确认弹窗
    
*   弹窗标题：提示
    
*   弹窗描述：移除白名单的对象，后续操作将执行风控判断逻辑。请确认是否删除白名单
    
*   取消按钮&关闭按钮：点击后关闭，无二次确认
    

**黑名单管理**

:::
主要调整：风控后台不再直接维护功能封禁能力，封禁能力统一收敛到封禁服务中

原因如下：

登录、注册、支付等关键链路已有实时风险打分能力，高风险行为可在当次请求中实时拦截。

实时风控适合处理即时行为风险，封禁管理适合处理确认后的长期对象限制，两者分层更合理。

风控侧单独维护黑名单会与封禁管理重复建设，黑名单应统一归入封禁管理。

统一封禁管理有利于查询、解除、审批、审计和跨业务系统接入。
:::
